##

2.cpu, memory 사용을 제한하는 LimitRange 생성
```
cat <<EOF | kubectl -n low-usage-limit create -f -
apiVersion: v1
kind: LimitRange
metadata:
  name: low-resource-range
spec:
  limits:
  - default:
      cpu: 1
      memory: 500Mi
    defaultRequest:
      cpu: 0.5
      memory: 100Mi
    type: Container
EOF
```

##

3.LimitRange 확인
```
kubectl get limitrange
kubectl get LimitRange --all-namespaces
```
