# Exercise 8.2

>root 관리자 계정으로 진행
```
sudo -i
```
##

1.NFS 서버 설치
```
sudo apt update && sudo apt install -y nfs-kernel-server
```

##

2.공유 디렉토리 생성 및 파일 생성
```
sudo mkdir /opt/sfw
sudo chmod 1777 /opt/sfw/
sudo bash -c 'echo software > /opt/sfw/hello.txt'
```

##

3.NFS 설정 파일에서 위에서 생성한 디렉토리 추가
```
sudo echo '/opt/sfw/ *(rw,sync,no_root_squash,subtree_check)' > /etc/exports
```


##

4.NFS 설정 반영
```
sudo exportfs -ra
```

##

5.Worker 노드로 이동 하여 NFS 설치
```
sudo apt-get -y install nfs-common
```
```
showmount -e k8scp
```
```
sudo mount k8scp:/opt/sfw /mnt
```
```
ls -l /mnt
```

##

6.CP 노드로 이동 하여 pv 생성
```
cat <<EOF | kubectl create -f -
apiVersion: v1
kind: PersistentVolume
metadata:
  name: pvvol-1
spec:
  capacity:
    storage: 1Gi
  accessModes:
    - ReadWriteMany
  persistentVolumeReclaimPolicy: Retain
  nfs:
    path: /opt/sfw
    server: k8scp
    readOnly: false
EOF
```

7.생성된 pv 확인
```
kubectl get pv
```
