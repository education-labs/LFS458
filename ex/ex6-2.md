# Exercise 6.2


1.프록시 명령어 도움말 확인
```
kubectl proxy -h
```

##

2.API prefix 설정하며 프록시 시작
```
kubectl proxy --api-prefix=/ &
```

##

3.curl 을 사용하여 api 접근
```
curl http://127.0.0.1:8001/api/
```

##

4.curl 을 사용하여 namespace로 접근
```
curl http://127.0.0.1:8001/api/v1/namespaces
```
